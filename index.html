Pig Dice

Rules:

  • Each player rolls the dice in turns
  • The program should return a random number from 1-6 when the roll dice button is clicked
  • It will not return a score if the dice returns 1 as the value
  • It will return equal to the value if the number is between 2
  • The program will add the scores to bring a total of 100
